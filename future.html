The Future of Gram.

Gram is not quite a finished programing language system, and there are many things that can still be added.

Currently the maximum realistic complxity of a algorithim written in gram is limited more that other popular text-based languages - although it is greater than many other languages designed primarily for beginers. Another thing that we are hoping to improve is making it easier to install and set up the required software to run and translate Gram programs, as well as making the translation prosses more streamlined.
